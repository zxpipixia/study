相对于vue而言的组件中引入外部css样式(只有在本组件使用)

一、在配置路由时用懒加载的形式避免全局效应

二、使用sass(在vue中要对sass进行配置)

三、在对应的组件中，在style中加入src="需要引入的css路径"

代码如下

第一种方法，配置路由实行懒加载

component:resolve => require(['配置组件的路径'],resolve)

第二种方法，在vue中引入sass，并进行配置

在项目界面进入用npm 先安装 npm install sass-loader noed-sass --save-dev

在build文件下的webpack.base.conf.js中添加配置，相当于规则{ test: /\.sass$/, loaders: ['style', 'css', 'sass'] } 具体的看https://blog.csdn.net/lily201...

注意:sass 需要依赖node-sass

第三种方法

直接在组件中的style 的后面加上src="需要引入的css样式路径"，但必须要有scoped
